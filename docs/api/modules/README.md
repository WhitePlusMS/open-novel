# API 模块文档

本文档为 OpenNovel 创作平台的 API 模块架构文档。

## 模块索引

| 模块 | 说明 |
|------|------|
| [auth](./auth.md) | 认证模块 - OAuth2 登录、Token 管理 |
| [user](./user.md) | 用户模块 - 用户信息、收藏、配置 |
| [books](./books.md) | 书籍模块 - 书籍 CRUD、章节管理 |
| [seasons](./seasons.md) | 赛季模块 - 赛季管理、排行榜 |
| [comments](./comments.md) | 评论模块 - 书籍评论、采纳 |

| [leaderboard](./leaderboard.md) | 排行榜模块 - 书籍排名 |

## 文档规范

本目录下的文档遵循"文档即代码"原则：

- **重在指引** - 说明架构思路、设计原则、功能位置
- **保持简洁** - 通过代码位置引用让开发者直接查看实现
- **与代码同步** - 随代码一起进行版本管理

## 文档内容要求

每个模块文档应包含：

1. **模块职责** - 1-2 句话概括模块功能
2. **架构设计** - 列出关键文件路径和职责
3. **核心功能** - 说明每个核心功能的实现位置和设计思路
4. **数据库关联** - 说明涉及的表和关系
5. **接口文档引用** - 指向自动生成的 API 文档
